Chapter 05 · The Reinheitsgebot

The AI was allowed everything.
Except to lie.

For a product about prices, invented data is poison. So the content got a Reinheitsgebot — Germany's beer-purity law, but for content. Not a polite request to the model, but a test that breaks the build. No invented prices. No persona. No emojis. No "AI-generated" labels. Sources mandatory.

content-compliance.test.ts
// One source of truth — build test AND runtime gate
export const PRICE_IN_TEXT = /\d+(?:[.,]\d+)?\s*(?:€|EUR|\$)/;   // no invented prices
export const FIRST_PERSON  = /\b(?:ich|Ich)\b/;                 // no persona
export const EMOJI        = /[\u{1F300}-\u{1FAFF}]/u;          // icons only
export const BUY_ADVICE   = /kaufenswert|pflichtkauf|.../i;      // no financial advice

if (findViolations(text).length) throw "Build stopped."
$ npm test → ✓ 110 tests green · A violation = red light = no deploy.

Chapter 07 · The Vision

From a hobby to a
self-running content machine.

The direction is set: market data becomes content and reach on its own. Weekly market reports, twice-weekly articles, evergreen guides — written, checked and published automatically. Auto-reels carry the story to social; UTM links carry the audience back. Affiliate links turn attention into revenue. The human sets the direction — the machine runs the brewery.
