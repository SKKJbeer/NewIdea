Kapitel 04 · Das Reinheitsgebot

Die KI durfte alles.
Außer lügen.

Bei einem Produkt über Preise ist Erfundenes Gift. Also bekam der Content ein Reinheitsgebot — nicht als Bitte an das Modell, sondern als Test, der den Build bricht. Keine erfundenen Preise. Keine Persona. Keine Emojis. Keine „KI-generiert"-Etiketten. Quellen Pflicht.

content-compliance.test.ts
// Eine Quelle der Wahrheit — Build-Test UND Laufzeit-Gate
export const PRICE_IN_TEXT = /\d+(?:[.,]\d+)?\s*(?:€|EUR|\$)/;   // keine erfundenen Preise
export const FIRST_PERSON  = /\b(?:ich|Ich)\b/;                 // keine Persona
export const EMOJI        = /[\u{1F300}-\u{1FAFF}]/u;          // nur Lucide-Icons
export const BUY_ADVICE   = /kaufenswert|pflichtkauf|.../i;      // keine Anlageberatung

if (findViolations(text).length) throw "Build gestoppt."
